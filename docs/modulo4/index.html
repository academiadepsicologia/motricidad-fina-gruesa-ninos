Módulo 4: Evaluación y Seguimiento

En este módulo aprenderás cómo observar y registrar el progreso de los niños en sus actividades de motricidad fina y gruesa.

¿Cómo evaluar?

  • Observa si el niño realiza la actividad solo o necesita ayuda.
  • Fíjate si mejora con la práctica: ¿lo hace más rápido? ¿con más precisión?
  • Pregunta al niño cómo se siente haciendo la actividad.
  • Registra el progreso regularmente para identificar patrones de desarrollo.

Consejo profesional

La evaluación debe ser un proceso continuo y natural, no un examen. Observa durante el juego y las actividades cotidianas para obtener información más auténtica.

Actividad práctica

Registro de avances

Esta actividad te ayudará a crear un sistema sencillo para registrar y seguir el progreso del niño en diferentes habilidades de motricidad.

  • 1
    Haz una tabla sencilla con el nombre de cada actividad y la fecha.
  • 2
    Marca con una carita feliz 😊 si el niño la hizo bien, una carita seria 😐 si necesita más práctica.
  • 3
    Añade notas breves sobre dificultades específicas o logros importantes.
  • 4
    Repite las actividades cada semana y observa el cambio en las marcas y notas.

Señales de alerta

Motricidad fina

  • Dificultad persistente para sostener utensilios
  • No puede recortar con tijeras a los 4 años
  • Problemas para abrochar botones
  • Dificultad para manipular objetos pequeños

Motricidad gruesa

  • Dificultad para saltar o correr
  • Tropieza con frecuencia
  • No puede subir escaleras alternando pies
  • Problemas de equilibrio constantes

Cuándo consultar a un especialista

Si observas varias de estas señales de alerta de forma persistente, es recomendable consultar con un especialista en desarrollo infantil o terapeuta ocupacional.
