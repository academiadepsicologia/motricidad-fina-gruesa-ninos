Módulo 1: Introducción a la Motricidad

¿Qué es la motricidad fina y gruesa?

La motricidad fina implica movimientos pequeños y precisos, como escribir o abotonar una camisa. La motricidad gruesa involucra movimientos amplios como correr, saltar o lanzar.
