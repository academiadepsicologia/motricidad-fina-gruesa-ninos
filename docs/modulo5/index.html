Materiales recomendados

Motricidad fina

  • Cuentas de diferentes tamaños
  • Hilos y cordones gruesos
  • Tijeras de seguridad
  • Plastilina o masa para modelar
  • Papeles de diferentes texturas

Motricidad gruesa

  • Almohadas y cojines
  • Pelotas de diferentes tamaños
  • Aros y conos
  • Cuerda para saltar
  • Colchoneta o alfombra gruesa
